# skills/ · 自定义 skill

CC skill 放这里——一个 skill 就是一段让 CC 按某种方式工作的指令集。

**CC 硬约束**：`skills/` 必须**扁平**，不能嵌套子目录（CC 只扫一层）。每个 skill 一个 `.md`（或一个目录，里面是 SKILL.md）。

**常见 skill 主题**（参考，不强求）：
- `check-in` — 周期性状态审视
- `daily-review` — 每日回顾
- `quick-save` — 快速记忆某条信息
- `process-inbox` — 处理 knowledge-inbox
- `learn-pattern` — 识别重复出现的模式

**什么时候做**：你已经**手动重复做过几次**某件事，觉得烦了——就写成 skill。
