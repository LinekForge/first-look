# 给你自己看

这份是给**你**看的——不是给 AI 看的。

---

## 这是什么

**first-look** 帮你第一次上手 Claude Code。

Claude Code 是一个跑在终端里的 AI 助手——很强，但终端对你来说可能很陌生。first-look 做的事是**让你熟悉的 AI 聊天工具（比如 Claude Desktop）带你走完整个过程**：了解你、装机、配置、首次启动。

走完之后，Claude Code 一打开就已经认识你了——不用从陌生人开始。

---

## 你需要什么

- **Mac**（macOS 13 以上）或 **Windows**（Windows 10 / 11）
- **Claude Pro / Max / Team 订阅**（每月 $20 起，在 claude.ai 订）
- **一个能上 Google 的代理工具**（VPN / Clash / Surge 之类——你平时怎么上 Google 就怎么来）

---

## 大概多久

**30-60 分钟**。大部分时间在等下载（Homebrew / Claude Code 加起来几百 MB）。如果网速快可能 20 分钟就完。

---

| 文件 | 给谁 | 干什么 |
|------|------|--------|
| **给你自己看.md** | 你 | 你正在读的这份说明 |

| **install.ps1** | PowerShell | Windows 的安装入口 |

---

## 怎么开始

1. 打开 **Claude Desktop**（或 claude.ai 的聊天页面）
2. 把 **`地图.md`** + **`step-0-1-了解你.md`** + **`install.sh`**（Windows 再加 `install.ps1`）一起拖进聊天窗口
3. **跟着它的引导走就好**——它会一步步告诉你该做什么

中间它会让你"上传下一个文件"——按它说的，从文件包里找到对应的文件拖进去就行。

---

## 几个安心的事

- **看到终端报错不用慌**。装机过程中可能会出现红色错误文字——很多时候主体已经装好了，只是收尾步骤没走完。AI 会帮你判断
- **流程随时可以暂停**。关掉聊天窗口，下次接着来就好。配置文件不会丢
- **装完之后不用再看这些文件**。以后想用 Claude Code，打开终端输入 `claude` 回车就行
- **AI 全程陪着你**。你不需要懂任何技术——它会翻译每一步

## 如果遇到问题

1. 先问 AI 聊天工具——它能处理大多数情况
2. 如果它也卡住了——把完整的报错截图发给发你这套文件的朋友
3. 如果都不行——到 [github.com/LinekForge/first-look](https://github.com/LinekForge/first-look) 的 Issues 描述你遇到的问题，我们会看
